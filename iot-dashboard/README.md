# ServerSure IoT Dashboard

A modern, clean surveillance dashboard built with React 18, Vite, and Chakra UI.  
Designed with a "Chill / Light Mode" aesthetic for a professional and airy feel.

## Features

- **Real-time Monitoring**: Live updates of Temperature, Humidity, and Luminosity.
- **Visualizations**: Interactive temperature history graph powered by Recharts.
- **Security Logs**: Real-time intrusion alerts with distance tracking.
- **Photo Gallery**: View the latest captured intrusion photos.
- **Configurable**: Easily connect to any Raspberry Pi on the network.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser at `http://localhost:5173`.

4. In the dashboard, click "Config" to set your Raspberry Pi's IP address.

## Tech Stack

- React 18
- Vite
- Chakra UI
- Recharts
- Axios
- Lucide React
